O Problema nos oceanos

A poluição dos oceanos e mares, causada principalmente por resíduos plásticos e químicos, ameaça ecossistemas marinhos, a biodiversidade e a saúde humana.

Tecnologias Utilizadas

Html 5 | Css 3 | JavaScript | C++ | Python

Nosso Objetivo

Nosso objetivo com esta iniciativa é aumentar a fiscalização da qualidade das águas, medindo o ph de tais. Dessa forma identificando possiveis locais de despejo de lixo inadequados. Dessa forma assegurando uma maior qualidade de vida para frequentadores das áreas litorâneas e animais marinhos.
